# Personal Habit Tracker

Персональный трекер привычек и задач с системой уведомлений и повторения.

## Возможности

### Управление задачами
- Привычки с ежедневными счётчиками и целями
- Задачи простые чекбоксы для разовых дел  
- Запланированные дела с напоминаниями и повторением

### Умные напоминания
- Уведомления о запланированных задачах
- Оповещения о просроченных делах
- Гибкая система повторения: ежедневно, еженедельно, ежемесячно, ежегодно

### История и управление
- Просмотр всех задач по категориям
- Удобное удаление выполненных дел
- Автосохранение прогресса

## Технологии

- C# .NET Windows Forms
- JSON для хранения данных  
- Windows Toast уведомления

## Как использовать
1. Добавьте привычку - укажите цель на день
2. Запланируйте дела - установите время и повторение
3. Отмечайте выполнение - кликайте на задачи
4. Следите за прогрессом - в реальном времени
